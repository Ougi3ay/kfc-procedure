# KFCProcedure Extended Documentation Package

This package contains a `docs/` folder and `mkdocs.yml` configuration for publishing developer documentation with MkDocs.

## Preview locally

```bash
pip install mkdocs mkdocs-material pymdown-extensions
mkdocs serve
```

## Build static site

```bash
mkdocs build
```

## Use in repository

Copy the `docs/` folder and `mkdocs.yml` into the root of the `kfc-procedure` repository.
