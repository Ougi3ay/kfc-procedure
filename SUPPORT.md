# Support

This page explains how to get help with `kfc-procedure`.

## Documentation

Start with the official documentation:

```text
https://ougi3ay.github.io/kfc-procedure/
```

The README also contains installation instructions and quick-start examples:

```text
https://github.com/Ougi3ay/kfc-procedure
```

## Questions

For general questions, use GitHub Discussions if enabled. If Discussions are not enabled, open an issue and choose the most relevant issue template.

## Bug reports

For reproducible bugs, open a GitHub issue using the bug report template. Include:

- package version,
- Python version,
- operating system,
- input shape and task type,
- minimal reproducible code,
- traceback or error message.

## Feature requests

For new functionality, open a feature request and describe the use case. Large API changes should be discussed before implementation.

## Security reports

Do not open public issues for security vulnerabilities. Follow `SECURITY.md`.

## Scope of support

This project can provide support for:

- package installation,
- documented APIs,
- reproducible bugs,
- documentation issues,
- contribution workflow,
- release and packaging questions.

This project does not guarantee support for:

- private datasets,
- production deployment consulting,
- unrelated machine learning projects,
- undocumented internal APIs.
